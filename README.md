## 🔧功能

- 支持2FA登录校验，登陆更快捷
- 最基本的电台日志记录
  - 频率管理、通联日志管理、日志导出等等
- 通联大屏，方便可靠直观发现通联数据
- 支持 ADIF 日志导入及导出，方便快速对接系统
- 支持关联 LoTW 系统，方便快速导入日志
- 支持多平台部署，**后续可能会考虑采用SaaS多用户模式，敬请期待**
- ~~支持客户端/服务端日志通联，用户可自行下载客户端添加日志至服务端~~~~
